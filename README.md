# Nijack Property Solutions

A Vercel-ready Vite + React + Tailwind starter for your wholesale real estate website.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy to Vercel

1. Upload this folder to GitHub.
2. Import the repo into Vercel.
3. Vercel should detect Vite automatically.
4. Add your custom domain in Vercel.

## Customize

Edit these files:

- `src/App.jsx` for page content
- `src/index.css` for global styles
- `index.html` for site title and meta description
